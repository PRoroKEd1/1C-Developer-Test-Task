# Тестовое задание 1С-разработчик

В репозитории представлена внешняя обработка `РаботаСВедомостями.epf`, реализующая логику тестового задания.

**Учтенные динамические условия (вариант выполнения):**
1. **Дата отправки (1-10 число):** В запросе реализовано условие отсева ведомостей, если день даты документа совпадает с днем текущей даты.
2. **Пол (Мужской):** Условное оформление (светло-зеленый цвет для статуса "Акцептован") реализовано строго программно на сервере с использованием компоновки данных.
3. **Длина фамилии (10 букв):** Информирование об удаленных ведомостях выводится не стандартным окном предупреждения, а текстом в специальный элемент основной формы обработки.
